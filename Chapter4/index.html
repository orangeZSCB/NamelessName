第四章 - 夜晚

直到推博不再弹出任何通知后，她才放心地关上推博。

她打开了 QQ——对她来说，推博用于发疯，QQ 才是她真正的跟别人倾诉所有的地方。

她一进 QQ，就看到一个群正在讨论些什么。成橙点了进去，但看到了她不想看到的东西。

A: 有人吃碳酸锂，25t！

（碳酸锂，抗躁狂药物，国家二级管制药物，过量服药导致癫痫致死。请按医嘱服药，切勿过量服药。笔者注。）

B: 啊？又是谁？

C: 不会又是 N732FE 吧？

B: 她怎么回事啊？！

A: 已经报警了，请大家少安毋躁。最近几天 N 老师状态不太好，据说家里人打算把她送去扭转。

C: 这是什么家长啊…

B: 希望没事…

N 老师是之前成橙在推博认识的。但真正的相识还是在另一个朋友的小群里。一开始还好好地一起聊天，直到有一天，N 老师的一句话打破了平静。

N732FE: 我炸柜了。

大约是在去年八月吧。

炸柜，是所有跨性别者的梦魇。——像是在玩扫雷，你永远不知道哪个格子，什么时候触到雷，再被炸得体无完肤。他/她们有的被朋友发现，或被安慰或被嫌弃，有的被学校发现，或被理解或被开除。最可怕的是被家长发现——或被爱，或被囚禁，断网，没收药物，折磨，送进医院看管或者扭转机构。扭转机构最为恐怖，不仅会被生理折磨，更是会被心理折磨，堪称新式“杨永信教育”，去过那些机构的人出来，严重的会出现生理问题，心理也会有一堆问题。留院观察就是在医院里面待一段时间，但是要是运气不好的话——就永远都出不来了，直到家长认为满意。

现在 N 老师这种企图了结自己的剧情，不仅在 N 老师身上上演过好几次，还在不同的人，不同的平台上上演过无数次。成橙想过帮忙，但什么都做不了。报警？警察回来找她的麻烦，就会炸柜。实地救援？一看就不现实，她还是初中生。没办法，成橙只能笨拙地复读那些安慰的话语，而结局也不会改变——什么实质性作用都没起到。

她合上电脑，趁父母不注意拿走了她自己的手机，回到屋中，掏出一颗提神用的橙子味霍氏薄荷糖，关上门和灯，装出一副已睡的样子，掏出手机打开 QQ 继续看。

A: 她父母是怎么知道我电话的？

B: 晶哥（警察 的别称），一定是晶哥又告诉家长了！

C: 完蛋了。

过了一段时间后。

C: 好，晚安。【一张图，桌子上摆放着亚硝酸钠（亚硝酸钠，一级致癌物，食品添加剂，也被用于少部分血管、支气管扩张药物中，过量后致死。请勿模仿。笔者注。）】

成橙不敢再看。成橙的大脑现在停止了运转。不知道是因为薄荷糖还是心理因素，她全身开始发冷了。

这下好了，警察告家长，N 老师又得住几天医院，又有一个人受 N 老师刺激自杀了。

成橙把聊天记录备份后删除了。她感到一种强烈的无力，伴随着悲哀，笼罩在心头。她什么都做不了。

她想着她与 N 的过往记忆，想到自己身上——自己炸柜了的话，她该怎么办？炸了柜会怎么样？是像 N 老师一样被折磨，还是在家长的支持下活出自己？

或者说…

自己，能活下来吗？

她也想过和身边的人出柜，坦白一切，但只有一个万年交的现实朋友是对此无任何评价的——既不喜也不悲，别人都表达了对成橙的厌恶… 最可惜的是和忧卡的三年友谊… “当他知道我吃那些药的时候他还在劝我不要乱吃… 我为什么要撒谎… 撒谎说自己不再吃了… 伤了他的心…” 她回忆起之前过量服药后的痛苦，回忆起那两个药，变得异常难受……

“铃，铃铃，铃，铃，铃，铃……”是成橙定的闹铃响了起来，用了她最爱的音游曲——sangkuli.uj 作的 Goodbye, Mr.Jack.

十一点了，

她知道，吃药时间又到了。
